# ASCII Filename Order

Let's see how files are sorted on GitHub or any other platform you like.

## ASCII Codes

FYI.

### 0x00 - 0x1F

Control characters

### 0x20 - 0x2F

Space, `!`, `"`, `#`, `$`, `%`, `&`, `'`, `(`, `)`, `*`, `+`, `,`, `-`, `.`, `/`

### 0x30 - 0x3F

0-9, `:`, `;`, `<`, `=`, `>`, `?`

### 0x40 - 0x4F

`@`, A-O

### 0x50 - 0x5F

P-Z, `[`, `\`, `]`, `^`, `_`

### 0x60 - 0x6F

`` ` ``, a-o

### 0x70 - 0x7F

p-z, `{`, `\|`, `}`, `~`, DEL
